# 출력 메서드

`Java`에서 콘솔에 출력하는 기능을 담은 3가지 메서드 비교

공통점: `System`이라는 클래스 사용. 이 클래스 안에서 `PrintStream`과 연결된 `out`이라는 필드를 사용함.

> 아래 3가지 메서드는 원래 `PrintStream`의 메서드임

&nbsp;

### 1. `print()`

출력 대상(기본형, `String`형 변수와 상수)을 콘솔에 출력함.

줄바꿈이 없기 때문에 연속적으로 사용하면 한 줄에 붙어서 나옴.

```java
System.out.print("안녕하세요.");
System.out.print("반갑습니다.");
// 출력 값:
// 안녕하세요.반갑습니다.
```

&nbsp;

### 2. `println()`

출력 대상(기본형, `String`형 변수와 상수)을 콘솔에 출력함.

자동으로 줄바꿈을 시켜주기 때문에 연속적으로 사용하면 줄마다 출력 결과가 나옴.

```java
System.out.println("안녕하세요.");
System.out.println("반갑습니다.");
// 출력 값:
// 안녕하세요.
// 반갑습니다.
```

&nbsp;

### 3. `printf()`

이름 끝의 'f'는 'format'의 의미로, 문자열 안에 다양한 지시자를 포함해서 그 값들을 유연하게 출력함.

자동 줄바꿈이 없으므로 줄바꿈 지시자(`%n`)을 사용해서 직접 줄을 바꿔줘야함.

```java
int count = 3;
System.out.printf("사과를 %d개 먹었습니다.", count);
// 출력 값:
// 사과를 3개 먹었습니다.
```
