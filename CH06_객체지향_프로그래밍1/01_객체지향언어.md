# 객체지향언어

## 객체지향의 기본 개념

"현실 세계는 사물(객체)로 이루어져 있고, 발생하는 모든 사건들은 사물(객체)들의 상호작용이다"

사물의 속성과 기능을 분석하여 컴퓨터 속에 옮겨 놓는 과정

- 사물 &#8594; 객체

- 속성 &#8594; 변수

- 기능 &#8594; 함수(메서드)

&nbsp;

## 객체지향언어

### 🛠 특징

1. 코드의 **재사용성**이 높음

   새로운 코드를 작성할 때 기존의 코드를 이용하여 효율적인 프로그래밍 가능

2. 코드 **유지보수**가 용이함

   코드 간의 관계를 이용해 효율적이게 코드 변경 가능

3. 높은 신뢰성

   제어자와 메서드를 이용해 데이터 보호, **중복된 코드 제거**
